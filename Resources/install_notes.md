# DSK SFz Player — Installation Notes

## English

### Requirements
- Windows 10 / 11 (64-bit)
- A VST3-compatible DAW **or** run the Standalone version directly

### VST3 Plugin

1. Copy **DSK SFz player.vst3** to your VST3 folder:

   `C:\Program Files\Common Files\VST3\`

2. Rescan plugins in your DAW (most DAWs do this automatically on next launch).

3. The plugin will appear under **DSK Music → DSK SFz player**.

### Standalone Application

1. Copy **DSK SFz player.exe** to any folder you like.
2. Double-click to launch — no installation required.

### First Use

- Click **+** in the Library panel to add a folder containing SFZ instruments.
- Or drag an **.sfz** file, a **folder**, or a **.zip** pack onto the plugin window.
- Click **Open SFZ / ZIP** to browse for an instrument directly.

### Uninstall

- VST3: delete `DSK SFz player.vst3` from the VST3 folder above.
- Standalone: delete the .exe file.
- Settings are stored in `%APPDATA%\DSK Music\DSK SFz player\`.
  Delete that folder to remove all saved preferences.

## Español

### Requisitos
- Windows 10 / 11 (64 bits)
- Un DAW compatible con VST3 **o** ejecuta directamente la versión Standalone

### Plugin VST3

1. Copia **DSK SFz player.vst3** en tu carpeta de VST3:

   `C:\Program Files\Common Files\VST3\`

2. Vuelve a escanear plugins en tu DAW (la mayoría lo hace automáticamente al siguiente inicio).

3. El plugin aparecerá bajo **DSK Music → DSK SFz player**.

### Aplicación Standalone

1. Copia **DSK SFz player.exe** en cualquier carpeta.
2. Haz doble clic para iniciarlo — no requiere instalación.

### Primer uso

- Haz clic en **+** en el panel de Librería para añadir una carpeta con instrumentos SFZ.
- O arrastra un archivo **.sfz**, una **carpeta** o un pack **.zip** sobre la ventana del plugin.
- Haz clic en **Open SFZ / ZIP** para buscar un instrumento directamente.

### Desinstalación

- VST3: elimina `DSK SFz player.vst3` de la carpeta VST3 indicada arriba.
- Standalone: elimina el archivo .exe.
- La configuración se guarda en `%APPDATA%\DSK Music\DSK SFz player\`.
  Elimina esa carpeta para borrar todas las preferencias guardadas.
